# ── Optional TTS cloud provider SDKs ────────────────────────────────────────
# These are NOT installed in the base Docker image because they add ~150MB
# and are only needed when the user selects that specific TTS provider.
# Each provider module uses a lazy import pattern (import inside function body)
# so the service starts fine without them.
#
# To use a cloud TTS provider, install the SDK at runtime:
#   docker exec voxwatch pip install elevenlabs   # ElevenLabs TTS
#   docker exec voxwatch pip install cartesia     # Cartesia Sonic TTS
#   docker exec voxwatch pip install boto3        # Amazon Polly TTS
#
# elevenlabs    # ElevenLabs TTS
# cartesia      # Cartesia Sonic TTS
# boto3         # Amazon Polly TTS
# kokoro-onnx   # Kokoro local TTS (install separately if using GPU)
